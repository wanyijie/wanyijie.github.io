##  我们已经学过如何用 which 来提问。

Which team has more fans?
哪个队有更多球迷？

Which 也可以用来把名词和修饰名词的词句连在一起！

That is a meter, which is about three feet.
那是一米，就是三英尺左右。

That is one foot, which is twelve inches.
那是一英尺，就是十二英寸。

This is one inch, which is about two and a half centimeters.
这是一英寸，就是两厘米半。
