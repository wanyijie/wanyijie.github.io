    Because vs Because Of

    leaning
    wangyijie, English

    在英语里，because 用在主句前面。 #

    I can’t see because it is so foggy. 因为雾太大了，我看不到。

    Because of 却会用在名词前面。

    It was difficult to see because of the snow. 因为雪，很难看清楚。

    Angela couldn’t sleep because of the noise. 因为那些噪音，Angela 没有睡觉。
