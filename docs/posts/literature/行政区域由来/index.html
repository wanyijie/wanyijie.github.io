    我们看电视，学历史，听过各种地域名字，各种郡，州 中华四个个地域划分方式的背景：

    1. 宗法分封制度（汉之前）
    2. 郡县制（郡是秦朝至唐朝的行政区划名）
    3. 州县制（至清）
    4. 省，市，县 九州：豫州、青州、徐州、扬州、荆州、梁州、雍州、冀州、兖州。 东南为扬州，越也。南方为荆州，楚也。西方为雍州，秦也。北方为幽州，燕也
