    Rds Mysql 快速结束进程

    RDS 默认管理员也没有super权限，数据库在执行DDL语句时没有super权限是不能结束进程会报异常：

     you are not owner for thread *******

    show 不是sql语句也不能处理输出，很多连接的时候我想到想到是使用excel来生成结束进程的kill命令，关键在于一个函数：在一单元格前后加上字符：

    ="kill "&B2&";"
